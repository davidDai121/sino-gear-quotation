# Sino Gear Quotation Project

This is a web-based quotation generator for Sino Gear.

## Project Structure

*   `public/`: Contains the source code.
    *   `index.html`: Main HTML file.
    *   `css/style.css`: Stylesheet.
    *   `js/script.js`: JavaScript logic.
    *   `assets/`: Folder for images and other assets.

## How to Start

1.  **Install dependencies** (optional, but recommended if you want to use the local server):
    ```bash
    npm install
    ```

2.  **Start the server**:
    ```bash
    npm start
    ```
    This will open the project in your default browser.

3.  **Alternative (Python)**:
    If you have Python installed, you can also run:
    ```bash
    python3 -m http.server -d public
    ```
